Course Overview

Locate any city, in any country across the globe with longitude, latitude, iso2, and iso3 numbers; each city displays the country name it is located. Simply register for an account, signin and start to learn. Users' can also add or remove either a city or country; this interactive solution is great for anyone that seeks to improve their knowledge of the locations of cities and countries across the globe. The solution consists of two projects; WorldCities and Health Check. If you clone this solution make sure set both projects to start simultaneously. Each application will open in their own browser window.
